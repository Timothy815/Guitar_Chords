# CAGED Teaching Panel Draft

**Purpose:** First-pass teaching copy for the `CAGED` page. This is the reference draft for validating the guided-learning panel pattern before broader rollout.

**Target UI:** One compact collapsible panel near the top of the `CAGED` page.

**Intent:** Help a user understand what CAGED is for, how to begin using this page, one common learning trap, and the best next step.

---

## Recommended Primary Copy

**Title**

`How to Use CAGED`

**Body**

`Learn the 5 movable chord shapes that map the neck. Start with one familiar shape, play it at a single root, then move it and listen for how the job stays the same while the location changes. Common trap: memorizing the box before noticing where the root lives inside it. Next: open Scale Positions and find the scale pattern that sits around the same area of the neck.`

---

## Why This Version

This version is written to match the current page:

- The page already supports selecting shapes and roots.
- It already encourages playing and hearing shapes.
- It naturally connects to `Scale Positions`.
- The main confusion point for most users is shape memorization without root awareness.

The copy aims to orient a user without repeating theory they may already know.

---

## Shorter Variant

Use this if the final UI feels crowded:

`CAGED shows the 5 movable chord shapes that organize the neck. Start with one shape at one root and listen for how the pattern moves without changing function. Common trap: learning the outline but not the root notes. Next: use Scale Positions to see the scale that lives around the same shape.`

---

## Longer Variant

Use this only if the final panel design comfortably supports more text:

`CAGED is a map of the neck built from 5 chord shapes you already know in open position. Start with one familiar shape at one root and move it slowly so your ear and hand learn the same chord function in new places. Common trap: memorizing the whole outline before you can point to the root notes inside it. Once the shape feels stable, open Scale Positions and find the scale pattern that surrounds the same chord area.`

---

## Copy Review Notes

What this draft intentionally does:

- gives a clear first move
- frames CAGED as a map, not just a set of grips
- emphasizes hearing and root awareness
- points to a concrete adjacent page

What it intentionally avoids:

- full theory explanation of all 5 shapes
- historical explanation of the acronym
- dense terminology
- telling the user to explore every control on the page

---

## Validation Questions

Before rolling this pattern out to other pages, confirm:

1. Does a new user understand what to do on `CAGED` after reading this once?
2. Does the panel feel helpful rather than beginner-only?
3. Does the next-step sentence make `Scale Positions` feel like a natural continuation?
4. Is the trap sentence specific enough to be worth keeping?

If the answer to any of these is no, revise this draft before using it as the tone standard for future guided-learning panels.
